ỨNG DỤNG RELAX APP - ALL IN ONE

Các chức năng chính:
 + Nghe nhạc mp3 online.
 + Xem Video online.
 + Đọc sách online.
 + Kho Audio book giúp người dùng học tập dễ dàng.
 + Cờ tướng.
 + Tic-tac-toe.
 + Ai là triệu phú.
 + Con số bí ẩn.
 + Camera Flashlight.
 + Tiền điện sinh hoạt 2019.
